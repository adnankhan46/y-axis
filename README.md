## Problem
While using LLMs, long conversations are hard to navigate, losing important prompts
scrolling endlessly to find one.

## Solution
[Y-Axis](https://yaxis.vercel.app) is a Chrome extension that builds a navigation layer
on top of LLM chats, allowing instant jumps across long conversations.
Saves a lot of time finding that one perfect prompt.

## Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl` + or `;` | Toggle Panel |
| `↑` / `↓` or `j` / `k` | Navigate Down/Up |
| `←` / `→` | Toggle View (Only Prompts/View All) |
| `Enter` | Jump to Message |
| `Esc` | Close Panel |
